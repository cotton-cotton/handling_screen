## 🥑 과제 개요

- 이벤트 감지에 따른 문구 내용 및 색상변경

## ✅ 과제 구현 목록

**_mouseenter_**

- 마우스 장치가 문구 위로 이동했을 때 이벤트 발생

**_mouseleave_**

- 마우스 장치가 문구 밖으로 이동했을 때 이벤트 발생

**_resize_**

- 화면 사이즈가 변경되었을때 이벤트 발생

**_contextmenu_**

- 마우스 오른쪽 메뉴 선택 시 이벤트 발생
